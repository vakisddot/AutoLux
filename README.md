# Autolux.bg

Note: This project is a mock demo only! The car data shown in the site is from a dataset. The images were fetched from Wikimedia's API.

## Commands
### Run
```bash
npm i    
npm run dev
```

### Build
```bash
npm i
npm run build
```

### Fetch car images
```bash
npm run fetch-images
```

## Stack
Astro, Tailwind, Three.js
